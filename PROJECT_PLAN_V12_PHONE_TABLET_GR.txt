PROJECT PLAN V12 — PHONE + TABLET FINAL

Στόχος:
Να μετατραπεί η V11 σε πιο πραγματική εφαρμογή για κινητό και tablet.

Κλειδωμένα σημεία:
1. Ακριβές logo του Ευάγγελου.
2. Responsive διάταξη για κινητό και tablet.
3. Κουμπιά/υπηρεσίες που ανοίγουν σωστά.
4. Εκπαιδευτική ύλη από Νηπιαγωγείο έως ΣΤ’ Δημοτικού.
5. Ορθογραφία, Σωστό/Λάθος, Κατανόηση, Υπαγόρευση demo.
6. Παραμύθια AI με επιλογές και ερωτήσεις κατανόησης.
7. Progress engine, levels, Smart Teacher feedback, επανάληψη λαθών.
8. AI Βιβλιοθήκη για γονέα/δάσκαλο με νόμιμο upload/επικόλληση κειμένου.
9. Γονικός πίνακας και School Mode preview.

Επόμενη εμπορική εξέλιξη:
- Πραγματικό backend και accounts γονέα.
- Πραγματική αποθήκευση παιδιών/προόδου.
- Πραγματικό AI API για δυναμικές ερωτήσεις και παραμύθια.
- Πραγματικός reader/OCR για PDF/βιβλία.
- Συνεργασίες με εκδοτικούς οίκους για νόμιμο παιδικό περιεχόμενο.
- PWA install / Android / iOS wrapper.
