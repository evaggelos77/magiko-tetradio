# Μαγικό Τετράδιο AI Junior

Παιδική AI εκπαιδευτική εφαρμογή για Νηπιαγωγείο έως ΣΤ' Δημοτικού.

- 👤 Στοιχεία γονέα / παιδιού + φωτογραφία παιδιού
- 🎒 Τάξεις & Ύλη · 🧠 Μάθημα · 📚 Παραμύθια AI · 📖 AI Βιβλιοθήκη
- 🌞 Ημερήσιο Πλάνο · 👨‍👩‍👧 Πίνακας Γονέα · 🏫 School Mode · 🛡️ Safe AI
- 📋 Οδηγίες λειτουργίας για γονείς

## Εγκατάσταση σαν εφαρμογή

Πατάς το κουμπί **⇩ Install** πάνω δεξιά. Σε iPhone Safari: «Κοινή χρήση» → «Προσθήκη στην οθόνη Αφετηρίας». Σε Android Chrome: μενού ⋮ → «Εγκατάσταση εφαρμογής».

## Τεχνικά

Static site (HTML/CSS/JS + Service Worker). Deploys στο Render ως Static Site χωρίς build step (βλ. `render.yaml`). Όλα τα δεδομένα παιδιού/γονέα μένουν τοπικά στον browser (localStorage) — δεν φεύγουν από τη συσκευή.
